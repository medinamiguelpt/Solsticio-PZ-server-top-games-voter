==========================================================
  ESPANOL - que hace cada boton
==========================================================

INSTALL.bat ......... Instala todo y te pregunta el idioma (solo una vez).
TEST-vote-now.bat ... Vota AHORA mismo.
                        - Si funciona: "VOTO CONFIRMADO" y el reloj se reinicia
                          (siguiente voto automatico en ~2h).
                        - Si ya votaste: "EN ENFRIAMIENTO" con el enlace para
                          ver cuanto tiempo falta.
SCHEDULE-every-2h.bat Vota ahora si se puede y activa el voto automatico
                        (luego cada ~2 horas).
STOP-scheduling.bat . Quita por completo el voto automatico.

PAUSE.bat ............. Pausa el voto automatico (sin borrarlo).
RESUME.bat ............ Reanuda el voto automatico y muestra la proxima vez.
STATUS.bat ............ Muestra si esta activo, ultima/proxima vez y el registro.

Recuerda: PC encendido y con sesion iniciada, y la VPN APAGADA.
Tu nombre se pone en  username.txt

==========================================================
  ENGLISH - what each button does
==========================================================

INSTALL.bat ......... Installs everything and asks your language (one time).
TEST-vote-now.bat ... Votes RIGHT NOW.
                        - If it works: "Vote CONFIRMED" and the timer restarts
                          (next automatic vote in ~2h).
                        - If you already voted: "ON COOLDOWN" with a link to
                          see how much time is left.
SCHEDULE-every-2h.bat Votes now if possible and turns on automatic voting
                        (then every ~2 hours).
STOP-scheduling.bat . Removes the automatic voting completely.

PAUSE.bat ............. Pauses automatic voting (without deleting it).
RESUME.bat ............ Resumes automatic voting and shows the next run time.
STATUS.bat ............ Shows if it's active, last/next run, and the log.

Remember: PC on and LOGGED IN, and VPN OFF.
Your name goes in  username.txt
